隐私政策

更新日期：2026年1月27日
重要提示：我们非常重视您的隐私保护。本隐私政策旨在向您说明我们如何收集、使用、存储和保护您的个人信息。请仔细阅读本政策，了解我们对您信息的处理方式。

一、引言

私密相册（以下简称"本应用"）是一款专注于保护用户隐私安全的相册管理应用。我们承诺以最小化、最安全的方式处理您的个人信息。

本应用的核心设计理念是：您的数据仅存储在您的设备本地，不会上传到任何服务器。

二、信息收集

1. 我们收集的信息

本应用在您使用过程中可能收集以下信息：

  • 媒体文件：照片、视频等媒体内容，存储在您的设备本地加密沙箱中
  • 设备信息：设备型号、操作系统版本（用于功能适配和兼容性优化）
  • 使用数据：应用崩溃日志、功能使用统计（用于改进产品质量）

2. 我们不收集的信息

  • ❌ 不收集您的姓名、手机号、邮箱等个人身份信息
  • ❌ 不收集您的位置信息
  • ❌ 不收集您的联系人信息
  • ❌ 不收集您的通讯录或短信内容

三、权限使用说明

本应用在以下功能中需要访问您的设备权限：

权限名称 使用场景 使用目的
相册访问权限 导入/导出照片和视频 将系统相册中的媒体文件导入到私密相册，或将私密相册中的文件导出到系统相册
相机权限 拍照、录像 直接拍摄照片或视频并保存到私密相册，无需经过系统相册
Face ID / Touch ID 生物识别解锁 使用生物识别快速验证身份，解锁应用（需您主动开启）
本地网络权限 DLNA 投屏 发现同一 Wi-Fi 网络下的投屏设备（电视、音箱等）
权限说明：以上权限均为可选，仅在您使用对应功能时才会请求。您可以随时在系统设置中关闭这些权限，关闭后不影响应用其他功能的使用。

四、信息存储与安全

1. 本地加密存储

本应用采用AES-256 加密算法对您存储在私密相册中的所有媒体文件进行加密保护。加密密钥基于您设置的密码生成，存储在设备的安全存储区域（Keychain）中。

这意味着：

  • ✅ 即使设备丢失，他人也无法访问您的私密文件
  • ✅ 文件仅在您的设备本地存储，不会上传到云端
  • ✅ 我们、运营商、应用商店都无法查看您的私密内容

2. 安全措施

我们采取了以下安全措施保护您的信息：

  • 密码强度要求：密码至少 4 位，建议使用复杂密码
  • 错误锁定机制：连续输入错误密码 5 次将锁定 30 秒，防止暴力破解
  • 生物识别支持：支持 Face ID / Touch ID 快速验证
  • 防截屏保护：应用界面禁止截屏录屏（iOS 系统限制）
  • 自动锁定：应用切换到后台或关闭屏幕后，需要重新验证身份

五、信息使用

我们使用收集的信息用于以下目的：

  • 提供核心功能：存储、管理、查看您的媒体文件
  • 身份验证：通过密码或生物识别验证您的身份
  • 功能改进：分析崩溃日志和使用数据，优化产品体验
  • 客户支持：在您联系我们时，帮助定位和解决问题

我们不会将您的信息用于以下目的：

  • ❌ 向第三方出售、出租或交易您的个人信息
  • ❌ 向第三方共享您的媒体内容
  • ❌ 用于定向广告或营销推广

六、数据共享与转让

除以下情况外，我们不会与任何第三方共享您的个人信息：

  • 征得您的同意：在获得您明确同意的情况下
  • 法律法规要求：根据法律法规或政府主管部门要求
  • 保护权益：为维护我们的合法权益，如防范欺诈行为

七、DLNA 投屏功能

本应用支持将视频投屏到同一 Wi-Fi 网络下的 DLNA 设备（如智能电视）。在此过程中：

  • 应用会在您的设备上启动一个临时的本地 HTTP 服务器
  • DLNA 设备通过本地网络访问该服务器获取视频文件
  • 视频内容不会经过任何云端服务器，仅在局域网内传输
  • 投屏结束后，本地服务器会自动关闭

八、应用伪装功能

本应用支持将应用图标更换为计算器、日历、闹钟、天气等常见应用图标，以更好地保护您的隐私。该功能：

  • 仅在您的设备本地更改应用图标
  • 不会影响应用的正常功能
  • 您可以随时恢复为默认图标
  • 我们不会因此收集或上传任何额外信息

九、无痕拍摄功能

开启无痕拍摄后，拍摄完成后应用会自动退出并清除最近任务记录。该功能：

  • 照片/视频直接保存到加密的私密相册
  • 不会在系统相册中留痕
  • 可配合应用伪装功能，进一步保护隐私

十、数据删除

您可以随时通过以下方式删除您的数据：

  • 删除单个文件：在应用中选择相应文件并删除
  • 批量删除：使用批量操作功能删除多个文件
  • 一键导出后删除：导出所有文件到系统相册的同时删除私密副本
  • 卸载应用：卸载应用将永久删除所有数据（无法恢复）
重要提示：删除操作不可逆，请谨慎操作。建议在删除前使用"一键导出"功能备份重要文件。

十一、未成年人保护

本应用主要面向成年人用户。如果您是未满 18 周岁的未成年人，请在监护人的指导下使用本应用。我们不会故意收集未成年人的个人信息。

十二、隐私政策的变更

我们可能适时修订本隐私政策。变更后的政策将通过应用内公告、官网公告等方式通知您。重大变更将通过弹窗等方式获得您的明确同意。

建议您定期查看本政策，以了解我们如何保护您的信息。

十三、您的权利

根据适用的法律法规，您对自己的个人信息享有以下权利：

  • 访问权：查看我们存储的关于您的信息
  • 更正权：要求更正不准确的信息
  • 删除权：要求删除您的个人信息
  • 撤回同意：撤回之前给予的同意（如关闭生物识别）
  • 注销账号：通过删除应用数据等同于注销账号
